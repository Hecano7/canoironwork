Gate Operator Installation

For all gate operators the client has to provide power to the location where the gate operator will be mounted. Depending wether it is a dual gate opener there has to be a pvc conduit under the driveway reaching both sides to each of the gate operators. For Swing gate application the cement columns has to be structurally sound to support the weight of the gates. For a Sliding gate application there will be a rail called a v-track on which the gate will roll on and will be bolted on the ground spanning twice the length of the gate. All new gate operators should come with a photo eye as a safety precaution to protect against any obstruction blocking the closing path of the door.
